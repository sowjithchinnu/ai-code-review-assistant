# AI Code Review Assistant

An AI-powered full-stack application that analyzes source code using static analysis and LLMs to provide intelligent code review, bug detection, performance suggestions, and best practices.

## Tech Stack

- Next.js
- Express.js
- PostgreSQL
- Tailwind CSS
- JWT Authentication
- ESLint
- OpenAI/Groq API

## 🚀 Features Implemented

### 📤 Code Submission
- Paste source code
- Upload source code files
- Secure file handling with Multer

### 🔍 Static Code Analysis
- Syntax error detection
- Unused variable detection
- Missing import detection
- Duplicate code detection
- Poor formatting detection (Prettier)
- Code style violation detection (ESLint)

### 🤖 AI Code Review
- AI-generated bug reports
- Optimization suggestions
- Code smell analysis
- Performance improvement recommendations
- Security recommendations
- Best practice suggestions
- Improved code generation using Groq Llama 3.3

### 📊 Complexity Analysis
- Cyclomatic Complexity
- Function Complexity
- File Complexity
- Number of Functions
- Number of Classes
- Lines of Code
- Duplicate code metrics

### 📚 Automatic Documentation
- Code summary generation
- Function documentation
- Class documentation
- API endpoint documentation
